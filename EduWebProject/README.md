Проект находится на стадии разработки (тесты, обработка ошибок увеличение функционала)

В этом проекте используется Spring (Boot, MVC, JPA)

Цель приложения: создать поиск косметики по списку ингредиентов. Продукты, по которым осуществляется поиск, предварительно загружаются в базу данных пользователем
