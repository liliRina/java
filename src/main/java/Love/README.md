В данном проекте используется Java Swing и демонстрируются принципы полиморфизма и наследования объектов.

Класс Отношение представлен 3 типами: Любовь и Ненависть (реализуют интерфейс Степень), Нейтральный.  

Принцип работы:
1. Выбирается тип отношений. 
2.1. Если выбран Нейтральный — выводится соответствующий текст-ответ. Программа заканчивается.  
2.2. В двух других случаях — на интерфейсе появляется возможность выбрать Степень (по умолчанию 0).   
3. В зависимости от выбранного типа и Степени выводится конечный текст.
