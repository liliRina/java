В данном проекте используется Java swing и демонстрируется полиморфизм объектов.

Класс Отношений представлен 3 типами: Любовь и Ненависть (имеют степень), Нейтральный.
В зависимости от выбранного типа либо выводится текст (для Нейтральный), либо предлагается выбрать степень (оставшиеся). В зависимости от степени и типа выводится текст
