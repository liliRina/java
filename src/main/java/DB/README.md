В этом проекте используются Hibernate, Hikari и JPA для работы с базами данных. JUnit - для создания тестов. Lombok - для сокращения шаблонного кода. java.util.concurrent - для многопоточного программировани.

Проект содержит несколько частей. 
1. Данные. 2 сущности: Колобки (встроенный класс - Тесто) и Калории.
2. Репозитории. Абстрактынй репозиторий и 2 для каждой сущности. В них происходит непосредственная работа с базой данных.
3. Класс для создания и настройки соединения с базой данных
4. Main

Тесты включают 2 файла:
1. RepositoryTest. Проверяется создание таблиц и связи между.
2. RunningsTest. Помимо использования базы данных, тут тестируется различный функционал для корректной работы с потоками (атомарные/неатомарные операции, инструменты управления потоками и пример deadlock для базы данных). Также проверяется, что hikari действительно создаёт максимально 10 потоков, как и указано в конфигурации.
