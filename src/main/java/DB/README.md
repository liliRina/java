В этом проекте используются:  
Hibernate, Hikari и JPA для работы с базами данных. 
JUnit - для создания тестов. 
Lombok - для сокращения шаблонного кода. 
java.util.concurrent - для многопоточного программирования.

Проект содержит несколько частей:
1. Данные. 2 сущности: Колобки (со встроенным классом - Тесто) и Калории.
2. Репозитории. Абстрактынй репозиторий и 2 его наследника для каждой сущности. В них происходит непосредственная работа с базой данных.
3. Класс для создания и настройки соединения с базой данных.
4. Main.

Тесты включают 2 файла:
1. RepositoryTest. Тестируется создание таблиц и связи между.
2. RunningsTest. Комплексные тесты, которые включают:
2.1. Тестирование работы с базой данных.
2.2. Сравнение атомарных и неатомарных операций.
2.3. Использование различных инструментов управления потоками.
2.4. Пример deadlock в базе данных.
2.5. Проверка соответствия количества потоков из конфигураций hikari реальному количеству потоков.
